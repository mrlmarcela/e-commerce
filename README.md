This project was developed in order to study vue 3 with typescript.

Were used:

- Vue.js 3
- Typescript
- Bootstrap
  

## Project setup
```
npm install
```

### To start the backend

```
npm run backend
```

### Compiles and hot-reloads for development
```
npm run serve
```
